Dealing with outliers¶

iskirtis turt bti iki 5% vis duomen. Jeigu daugiau, tai jau nebe iskirtis. Nustatyti iskirtis galima bdais:

  • klausti t duomen ekspert, kas yra iskirtis
  • jeigu normaliai pasiskirst (arba transformuoti norm.) tai pagal std. nuokryp |zi| > 3 (>2)
  • pagal IQR
  • irti kurie takai neseka bendros duomen tendencijos (tarp labiausiai koreliuot)

# Choose a mean,standard deviation, and number of samples

def create_ages(mu=50,sigma=13,num_samples=100,seed=42):
    np.random.seed(seed)

    #create random standart normal distribution of sample ages
    sample_ages = np.random.normal(loc=mu,scale=sigma,size=num_samples)
    sample_ages = np.round(sample_ages,decimals=0) # round, so there are no decimals
    
    return sample_ages # return integers

# atfiltruojam kur trksta ir kokia dalis trksta
def percent_missing(df):
    percent_nan = 100* df.isnull().sum() / len(df)
    percent_nan = percent_nan[percent_nan>0].sort_values()
    return percent_nan

Dirbam su trūkstamom kategorinėm reikšmėm¶

Gali bt kategorini stulpeli kurie paymti skaiiais, bet jeigu tos kategorjos nra rikiuojamos (ordinal) tai skaiiais jas ymti nra gerai. Modelis gali interpretuoti, kad vienos kategorijos geriau u kitas nors taip nra. Reikia pasirpinti, kad taip nebt duomen aibje. K galime padaryti dl to ? Galime sukurti dummmy variables t.y. kategorijas paversti stulpelius ir reikmse tsg ymti 1-yra,0-nra. Tai vadinama One-hot encoding. Jeigu labai daug gausis stulpeli, tai galima i pdradi stransformuot higher order kategorijas t.y. susiaurinti, pvz: salys -> emynai.
